DICTIONNAIRE FRANCAIS -> BANGANTE

Description:
Ce Mini Dictionnaire realiser en HTML, CSS et JS permet de comprendre la significations des mots francais en bangante qui est une langue maternelle de l`ouest .

HTML : permet l`ecriture du texte (dictionnaire, francais-> bangante, traduction....).

CSS: permet de styliser le dictionnaire (les couleurs de fond  melanges, la taille des textes, la dimension du dictionnaire, la couleur des textes (blanc, blue, jaune ...))

JS : permet de dynamiser le dictionnaire (faire que lorsqu` on clique sur Traduction le mot traduis  s`affiche en bas )

Json: contient les mots et leurs significations en bangante.
## Fonctionalites : 
Entrez le mot  dont vous voulez savoir la signification , cliquer sur Traduction puis la signification vas apparaitre en bas.
